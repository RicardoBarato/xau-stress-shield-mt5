# Continuation Guide

## Recommended protocol

1. Define a baseline strategy.
2. Define the data contract.
3. Define stress labels.
4. Build causal features only.
5. Define metrics and false-positive costs.
6. Run ablation tests.
7. Test multiple years.
8. Test multiple brokers.
9. Run Monte Carlo analysis.
10. Publish positive and negative outcomes together.

## Acceptance mindset

The shield should improve survival evidence, not hide a weak entry model. Operational approval must remain separate from research usefulness.
